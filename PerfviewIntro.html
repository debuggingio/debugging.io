perfview is a system wide profiler for quickly and easily collecting and viewing both time and memory performance data. It's capability understand events emitted by .net runtime makes this tool an easy companion for .net debugging

What can PerfView do for you?

  • CPU Investigation
  • Managed Memory Investigations
  • Wall Clock / Blocked Time Investigations
  • Unmanged Memory Investigations

Using the tool:Enviroment

  • PerfView itself is a .NET 4.0 application. Needs .NET 4.0+.
  • On 2008/2008 R2 .NET 4.0 is automatically installed via Windows Update.
  • 2012+ comes with .NET 4.0.
  • Not supported in XP, 2003.
  • Can collect information of .NET 2.0 application. Use other tools (xperf, or others) to capture, and then view it with PerfView.
