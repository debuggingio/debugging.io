REINTRODUCING ASP.NET CORE 1.0 AND .NET CORE 1.0

ASP.NET 5 is now ASP.NET Core 1.0.

.NET Core 5 is now .NET Core 1.0.

Entity Framework 7 is now Entity Framework Core 1.0 or EF Core 1.0 colloquially.

.NET Core

.NET Core is a subset of the .NET Framework which will run on Windows, Linux ,OS X and FreeBSD and completely open source.

https://github.com/dotnet/coreCLR

asp.net core

ASP.NET Core is a new open-source and cross-platform framework for building modern cloud-based Web applications using .NET core.

Foundational improvements of asp.net core

  • New light-weight and modular HTTP request pipeline
  • Ability to host on windows,linux,OS X or FreeBSD
  • Built on .NET Core, which supports true side-by-side app versioning
  • Ships entirely as NuGet packages
  • Single stack for Web UI(MVC) and Web APIs
  • Open source and community focused
